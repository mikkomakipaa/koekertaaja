## Visual Design Guidelines

### Colors
- **Primary CTA**: Emerald-600 (NEW - fresh, growth-focused green)
- **Quiz Mode**: Indigo (existing)
- **Flashcard Mode**: Teal (existing)
- **Subjects**:
  - Matematiikka: Blue
  - Ã„idinkieli: Rose/Pink
  - YmpÃ¤ristÃ¶oppi: Green
  - Englanti: Orange

**Color Harmony**: The emerald primary creates a cohesive cool-toned, nature-inspired palette that complements both indigo and teal. The green symbolizes growth, learning, and freshness - perfect for an educational app.

### Typography
- **Headlines (H1)**: 2xl-3xl on mobile, 4xl-5xl on desktop
- **Section Titles (H2)**: xl-2xl on mobile, 3xl on desktop
- **Body**: Base size, good line-height for readability
- **Feature Titles**: Semibold, sm-base

### Spacing
- **Sections**: 12-16 gap on mobile, 16-24 on desktop
- **Cards**: p-4 on mobile, p-6 on desktop
- **Consistent rhythm**: Use multiples of 4

### Icons
- **Phosphor Icons** (already in use):
  - Duotone weight for main features
  - Size 24-32 for feature icons
  - Size 16-20 for inline icons

### Cards/Sections
- Rounded-xl borders
- Subtle shadows on hover
- Dark mode support with dark:bg-gray-800 patterns

## Mobile-First Breakpoints

### Mobile (< 640px)
- Single column layout
- Stack all elements vertically
- Full-width buttons
- Accordion or stacked tabs for audience sections
- Comfortable touch targets (min 44px)

### Tablet (640px - 1024px)
- Two-column for mode comparison
- Grid for subject showcase (2 cols)
- Tabs for audience sections
- Larger CTAs

### Desktop (> 1024px)
- Three-column for some sections
- Side-by-side mode comparison with screenshots
- Horizontal tabs for audience sections
- Max-width container (4xl or 5xl)
