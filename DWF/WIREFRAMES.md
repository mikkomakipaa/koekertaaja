**Key Elements (Topic Selection)**:
- "All topics" option (default)
- Individual topic buttons with card counts
- Auto-select if 0-1 topics available

**Key Elements (Flashcard)**:
- Card number tracker
- Progress bar
- Topic label
- Question on "front"
- Answer input
- "Check answer" button
- Explanation on "back"
- "Next card" button

**Flashcard Results**:
- Similar to quiz results but simpler
- Only shows explanations, no scores
- Emphasizes learning over points

## Responsive Breakpoints

**Mobile (< 640px)**:
- Stack elements vertically
- Full-width cards
- Larger touch targets (min 48px)
- Hide less important metadata

**Tablet (640px - 1024px)**:
- 2-column layouts for cards
- Show more metadata
- Side-by-side buttons

**Desktop (> 1024px)**:
- 3-4 column layouts
- Maximum width containers (1280px)
- More compact layouts
- Show all metadata

---

## Navigation Patterns

**Primary Navigation**:
- Top: Back button (left), Page title (center)
- Bottom: Action buttons (full width on mobile)

**Secondary Navigation**:
- Footer: Links (sticky on long pages)
- Breadcrumbs: Not used (simple hierarchy)

**Modal Navigation**:
- Overlay with backdrop
- Close button (X) or "Cancel"
- Escape key to close
