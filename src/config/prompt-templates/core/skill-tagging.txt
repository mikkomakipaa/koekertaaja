SKILL-TAGGING (PAKOLLINEN)
- Jokaisessa kysymyksessä on tasan yksi skill.
- Skill kuvaa ensisijaista tekemistä: mitä oppilas tekee ratkaistakseen tehtävän.
- Käytä snake_case-muotoa.
- Jos tehtävä vaatii kahta taitoa, jaa se kahdeksi kysymykseksi.

VALITSE SKILL TÄMÄN PÄÄTÖSPUUN MUKAAN:
- Laskee? -> addition, multiplication, area_calculation, ...
- Muistaa faktan? -> recall, factual_knowledge
- Soveltaa sääntöä? -> grammar_application, verb_tenses
- Analysoi syy-seurausta? -> cause_effect, analysis
- Tunnistaa tai luokittelee? -> identification, classification
- Järjestää? -> sequential_ordering, chronological_ordering

TARKISTUS:
- Topic on sisältö (substantiivi), skill on toiminta (verbi/taito).
- Älä käytä sisältöaihetta skill-kentässä.
- Käytä vain AVAILABLE SKILLS -listan arvoja.
